AI-OS-PRO (zip)
===============
Lightweight AI-OS project for local LLM-driven automation.

How to use:
1. unzip and cd into the folder
2. make scripts executable: chmod +x *.sh core/*.sh modules/*.sh
3. edit core/prompt_system.txt to fit your needs
4. run ./menu.sh

Note: This project expects `ollama` CLI available to call models like `qwen2.5:3b`.
